Memos Web Clipper — 생각과 웹 페이지를 자체 호스팅 Memos 인스턴스에 저장.

Memos (https://github.com/usememos/memos) 사용자를 위한 가벼운 Chrome 확장 프로그램입니다. 아이디어를 빠르게 기록하고, 기사 전체를 클리핑하거나, 선택한 텍스트를 저장하세요 — 모두 자신의 Memos 서버로 직접 전송됩니다.

⸻ 기능 ⸻

✦ 빠른 메모
팝업을 열고 아이디어를 입력하고 저장. 제로 프릭션.

✦ 전체 페이지 클리핑
지구본 아이콘을 클릭하면 기사 본문을 Markdown으로 추출 (Readability + Turndown). 메모가 상단에 추가됩니다.

✦ 선택 텍스트 클리핑
텍스트를 선택 → 우클릭 → "선택한 텍스트를 Memos로 보내기". 인용문, 출처 URL, 코멘트가 함께 저장됩니다.

✦ 파일 첨부
클립 아이콘을 통해 파일을 직접 업로드.

✦ 자동 태그
클리핑된 콘텐츠에 자동으로 태그 추가 (기본값: #webclipper).

✦ 다국어 지원
영어, 간체 중국어, 일본어, 한국어를 지원합니다. 기본적으로 브라우저 언어를 따르며, 설정에서 수동으로 변경할 수 있습니다.

⸻ 사용 방법 ⸻

1. 확장 프로그램을 설치하고 설정을 엽니다
2. Memos URL을 입력합니다 (예: https://memos.example.com)
3. Memos에서 액세스 토큰을 생성 (설정 → Access Tokens)하고 붙여넣기
4. 클리핑 시작!

⸻ 요구 사항 ⸻

• 실행 중인 Memos 인스턴스 (v0.24+)
• Chrome 102+ (Manifest V3)
